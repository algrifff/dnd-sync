# Compendium

Self-hosted real-time vault for D&D, with character-level live cursors and a per-OS one-click installer for friends.

The whole project lives in [`compendium-ai/`](./compendium-ai). Start there:

- [`compendium-ai/README.md`](./compendium-ai/README.md) — quick start
- [`compendium-ai/ARCHITECTURE.md`](./compendium-ai/ARCHITECTURE.md) — design + roadmap

Older approaches (Syncthing on `main`, Self-hosted LiveSync on `livesync-couchdb`) are preserved on their respective branches if you ever want to roll back.
